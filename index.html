WHAT WE DO
Chainstamp gives you away to store a “fingerprint” of any digital file in a public ledger at a point in time, without exposing the actual file contents to anyone.
PERFECT FOR
From compliance, legal documents and contract signatures to creative works like photographs, music and manuscripts chainstamp is perfect.
BLOCKCHAIN PROOF
By being both immutable and beyond control of any company, blockchains guarantee your proof exists forever.

Putting the Blockchain to work

Our goal is to help businesses and individuals start to take advantage of this global, distributed, censorship resistant technology known as “blockchain”. It is time to start using it.

ROADMAP

Website Launch
Initial launch of chainstamp.io website will include creating timestamps on the bitcoin blockchain, letting users anywhere in the world create immutable records for any digital file.
API and Subscription Service
We will launch a public API and tools to make it easy for machine to machine chainstamps proofs to be created without any user interaction. Ideal for businesses, mobile apps, automation.
Integrations (email, social, etc.)
Email, Twitter integration as well as customized integrations for specific applications (e.g. dropbox, S3) will be launched.
Multiple blockchains
We will give customers options for creating timestamps in multiple blockchains.
Encrypted Distributed File Storage
This is great for people or businesses that need absolute privacy of data, while still allowing for cloud backups.
